########################### How to run the code. #########################################

1) To run a code first download Desai_jbd0245.zip and extract it.
2) Install pycharm from below link.
   CLICK HERE TO DOWNLOAD: https://www.jetbrains.com/pycharm/download/ 
3) Now download python from https://www.python.org/ and open environment and set path of python in  windows.
4) Open pycharm and go to settings and search for "python interpretor".
5) Click on " + " symbol and create a virtual environment.
6) Now open downloaded file(Desai_jbd0245) in pycharm.
7) Now open Server.py and go to run and click Run'Server'.
8) After that you will see message on console that server in running.
9) Now open Client.py and go to run and click Run'Client'. 
10) You do not need to explicitly give ip and port to server or host. It has already been given.
11) After opening Client.py run that file again and again to open more Client window.
12) log.txt file keeps track of the log so whenever server crashes it loads up past messages.
13) Install json , time , threading , socket before running the code.
